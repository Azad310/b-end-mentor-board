# 师兄认证看板 - 钉钉AI表格对接方案

## 现状
- 看板文件：`index.html`（纯前端，数据存在浏览器localStorage）
- 已有AI表格：**"B端师兄认证进度"**（baseId: `oP0MALyR8kzGnoOwFKkpnaqmJ3bzYmDO`）
- AI表格已有数据表：**"数据表1"**（tableId: `hERWDMS`）

## 改造目标
1. 看板从钉钉AI表格读取数据（替代localStorage）
2. 上传Excel后自动同步到AI表格
3. 多人共享，数据实时更新
4. 支持搜索、筛选、下载

## 实施方案

### 方案A：看板直接读取AI表格（推荐）

**原理**：钉钉AI表格支持"公开分享"功能，生成一个只读API链接，前端JS可以直接fetch获取JSON数据。

**步骤**：
1. 在钉钉AI表格中开启"公开分享"（只读）
2. 获取分享链接（形如 `https://alidocs.dingtalk.com/api/aitable/export/...`）
3. 看板JS通过fetch获取数据
4. 上传Excel后，调用dws命令同步到AI表格

**优点**：
- 无需后端服务器
- 多人实时看到最新数据
- 数据永久保存在钉钉云端

**需要你操作**：
1. 打开钉钉AI表格"B端师兄认证进度"
2. 点击右上角"分享"→"开启公开分享"→选择"只读"
3. 把生成的分享链接发给我
4. 我会把链接配置到看板的JS代码里

---

### 方案B：定时同步脚本（备用）

**原理**：写一个Node.js脚本，定期从Excel文件同步数据到AI表格。

**步骤**：
1. 把所有Excel文件放到固定文件夹
2. 运行同步脚本：`dws aitable import ...`
3. 看板从AI表格读取

**优点**：
- 不依赖公开分享
- 数据更新可控

**缺点**：
- 需要手动/定时运行脚本
- 实时性不如方案A
